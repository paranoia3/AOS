Correlation & Regression (Корреляция и регрессия)

When to use it

Эта тема — про связь между двумя переменными по выборке пар \( (x_{i},\ y_{i}) \).

  • Корреляция (correlation) измеряет, насколько линейно связаны \(x\) и \(y\), через коэффициент PMCC (product moment correlation coefficient) \(r\).
  • Регрессия (regression) — построение прямой \( y = a + bx \), которая лучше всего описывает данные, чтобы прогнозировать \(y\) по \(x\).

Ключевые слова в условии:

  • «calculate / interpret the product moment correlation coefficient» → корреляция;
  • «find the equation of the regression line of y on x», «estimate / predict y when x = …» → регрессия;
  • «using the coding p = (x − a)/b» — упрощение вычислений; \(r\) при кодировании не меняется;
  • «describe the correlation» — словесная интерпретация (positive / negative / no / strong / weak).

Definitions

Диаграмма рассеяния (scatter diagram) — точки \((x_{i},\ y_{i})\) на плоскости. По форме облака судят о наличии и характере связи.

Положительная корреляция (positive correlation) — \(y\) растёт с ростом \(x\) (точки в I и III квадрантах). Отрицательная (negative) — \(y\) убывает с ростом \(x\) (II и IV квадранты). Нет корреляции (no correlation) — точки во всех четырёх квадрантах без чёткого тренда.

Коэффициент корреляции PMCC (product moment correlation coefficient) \(r\) — число от \(-1\) до \(+1\), измеряющее силу и направление линейной связи: \(r = +1\) — точная положительная прямая; \(r = -1\) — точная отрицательная прямая; \(r = 0\) — линейная связь отсутствует.

Независимая переменная (independent / explanatory variable) — \(x\), задаётся / контролируется. Зависимая (dependent / response variable) — \(y\), её значения определяются \(x\). Регрессия \(y\) на \(x\) — прямая \( y = a + bx \), минимизирующая сумму квадратов отклонений.

Интерполяция (interpolation) — предсказание внутри диапазона данных (обычно надёжно). Экстраполяция (extrapolation) — за пределами диапазона (ненадёжно).

Key formulas

Формула 1 — Суммы \[ S_{xx} = \sum (x - \bar{x})^{2} = \sum x^{2} - \frac{(\sum x)^{2}}{n} \] \[ S_{yy} = \sum y^{2} - \frac{(\sum y)^{2}}{n}, \qquad S_{xy} = \sum xy - \frac{\sum x \sum y}{n} \]
Базовые величины, через которые считаются \(r\) и регрессия
Формула 2 — PMCC \[ r = \frac{S_{xy}}{\sqrt{S_{xx}\cdot S_{yy}}} \]
Коэффициент корреляции, \(-1 \le r \le 1\)
Формула 3 — Линия регрессии \[ y = a + bx, \qquad b = \frac{S_{xy}}{S_{xx}}, \qquad a = \bar{y} - b\bar{x} \]
Прямая «\(y\) на \(x\)», \(b\) — угловой коэффициент, \(a\) — отрезок на оси \(y\)
Формула 4 — Кодирование \[ \text{Если } p = \frac{x - a_{0}}{b_{0}},\ q = \frac{y - c_{0}}{d_{0}}, \quad \text{то } r_{xy} = r_{pq} \]
Линейное кодирование не меняет \(r\). Для регрессии — подставляем коды обратно

Interpretation of r

\(|r|\)Сила связи
\(0.0\text{–}0.3\)слабая (weak)
\(0.3\text{–}0.7\)умеренная (moderate)
\(0.7\text{–}1.0\)сильная (strong)

How to use it (step by step)

  1. Считаем суммы: \(\sum x,\ \sum y,\ \sum x^{2},\ \sum y^{2},\ \sum xy\) (либо берём их из условия / упрощаем кодированием).
  2. Считаем \(S_{xx},\ S_{yy},\ S_{xy}\) по Формуле 1.
  3. Корреляция: \(r = \dfrac{S_{xy}}{\sqrt{S_{xx}S_{yy}}}\); интерпретируй знак и величину.
  4. Регрессия \(y\) на \(x\): \(b = S_{xy}/S_{xx}\), \(a = \bar{y} - b\bar{x}\); запиши \(y = a + bx\).
  5. Кодирование: для \(r\) — переноси прямо; для регрессии — подставь \(x\) и \(y\) обратно в кодовое уравнение и упрости.
  6. Прогноз: подставь нужный \(x\) в \(y = a + bx\). Внутри диапазона данных — надёжно; вне — экстраполяция, ненадёжно.

Common mistakes

  • Корреляция ≠ причинность. \(r\) близкое к \(\pm 1\) означает только линейную связь, а не что \(x\) вызывает \(y\).
  • Регрессия \(y\) на \(x\) ≠ \(x\) на \(y\). Это разные прямые, нельзя одну использовать вместо другой.
  • Экстраполяция. Прогноз вне диапазона \(x\)-данных всегда оговаривай как ненадёжный.
  • Кодирование и регрессия. \(r\) не меняется при линейном кодировании, но коэффициенты регрессии — меняются. Не забыть перевести назад.
  • Нелинейные тренды. Малое \(|r|\) не значит, что связи нет — могла быть нелинейная (например, парабола).

Mini example

Пять точек: \( (1,2),\ (2,4),\ (3,5),\ (4,4),\ (5,5) \). Тогда \(n=5\), \(\bar{x}=3\), \(\bar{y}=4\),

\[ \sum x = 15,\ \sum y = 20,\ \sum x^{2} = 55,\ \sum y^{2} = 86,\ \sum xy = 66 \] \[ S_{xx} = 55 - \frac{15^{2}}{5} = 10,\quad S_{yy} = 86 - \frac{20^{2}}{5} = 6,\quad S_{xy} = 66 - \frac{15\cdot 20}{5} = 6 \] \[ r = \frac{6}{\sqrt{10\cdot 6}} = \frac{6}{\sqrt{60}} \approx 0.775 \quad (\text{умеренная–сильная положительная}) \] \[ b = \frac{6}{10} = 0.6, \quad a = 4 - 0.6\cdot 3 = 2.2 \;\Rightarrow\; y = 2.2 + 0.6x \]
