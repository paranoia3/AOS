a Scatter diagram. b Describe and interpret the correlation.

Решение (Solution)

b) Точки лежат почти вдоль прямой, идущей вверх вправо. Сильная положительная корреляция (\(r \approx 0.99\)). Чем длиннее поездка, тем выше плата — это логично для тарификации такси.

Corr — Problem 2

Условие (Problem statement)

Three scatter diagrams: i length vs age of snakes; ii % drop in wages vs % unemployment; iii men's height vs age. State whether each shows positive, negative or no correlation, and interpret.

Описать корреляцию и интерпретировать три графика.

Решение (Solution)

i) Положительная. С возрастом змеи в среднем длиннее — они растут.

ii) Отрицательная. Чем выше уровень безработицы, тем сильнее (в среднем) падают зарплаты — экономическая логика.

iii) Корреляция отсутствует. У взрослых мужчин рост уже не зависит от возраста.

Ответ

\[ \boxed{\text{i) positive} \quad \text{ii) negative} \quad \text{iii) no correlation}} \]

Corr — Problem 3

Условие (Problem statement)

Three scatter diagrams have PMCC values \(-0.12,\ 0.87,\ -0.81\). Match each value to a diagram with reason.

Сопоставить значения \(r\) трём диаграммам.

Решение (Solution)

Сопоставляем по виду:

  • i — точки идут вверх вправо, плотно у прямой → \(r = 0.87\) (сильная положительная).
  • ii — точки разбросаны без видимого тренда → \(r = -0.12\) (практически нет связи).
  • iii — точки идут вниз вправо → \(r = -0.81\) (сильная отрицательная).

Ответ

\[ \boxed{\text{i: } 0.87,\quad \text{ii: } -0.12,\quad \text{iii: } -0.81} \]

Corr — Problem 4

Условие (Problem statement)

PMCC for a person's age and memory test score is \(-0.86\). Interpret.

\(r = -0.86\) для возраста и результата теста памяти.

Решение (Solution)

Сильная отрицательная корреляция: чем старше человек, тем (в среднем) ниже результат теста памяти. Связь линейная и достаточно тесная (но не идеальная), и она не доказывает причинно-следственную связь — лишь устойчивую тенденцию.

Ответ

\[ \boxed{\text{сильная отрицательная — память снижается с возрастом}} \]

Corr — Problem 5

Условие (Problem statement)

10 people, Science \(s\), Art \(a\); coded \(x = s/10\), \(y = a/10\): \(\sum x = 67\), \(\sum x^{2} = 465\), \(\sum y = 65\), \(\sum y^{2} = 429\), \(\sum xy = 434\).

a Calculate PMCC for \(x,\ y\). b Write down PMCC for \(s,\ a\). c Are people good at Science also good at Art?

Найти \(r\) с кодированием; \(r\) без кодирования; вывод.

Решение (Solution)

\(n = 10\). Считаем суммы:

\[ S_{xx} = 465 - \frac{67^{2}}{10} = 16.1,\ S_{yy} = 429 - \frac{65^{2}}{10} = 6.5 \] \[ S_{xy} = 434 - \frac{67\cdot 65}{10} = 434 - 435.5 = -1.5 \]

a) \(r = -1.5/\sqrt{16.1\cdot 6.5} = -1.5/10.230 \approx -0.147\).

b) \(r\) для \(s, a\) — то же значение, кодирование не меняет: \(\approx -0.147\).

c) \(r\) очень слабое и отрицательное. Нет, нельзя сказать, что сильные в Science одновременно сильны в Art — связь между этими предметами в группе практически отсутствует.

Ответ

\[ \boxed{r \approx -0.147,\ \text{связи практически нет}} \]

Corr — Problem 6

Условие (Problem statement)

20 oranges: juice \(j\) (ml), price \(p\). \(\sum j = 979\), \(\sum p = 735\), \(\sum j^{2} = 52\,335\), \(\sum p^{2} = 32\,156\), \(\sum jp = 39\,950\).

a Find \(S_{jj},\ S_{pp},\ S_{jp}\). b Compute PMCC. c Describe correlation.

Апельсины: сок vs цена. \(S\) и \(r\). Прав ли Нимер?

Решение (Solution)

a) \(n = 20\):

\[ S_{jj} = 52\,335 - \frac{979^{2}}{20} = 4412.95 \] \[ S_{pp} = 32\,156 - \frac{735^{2}}{20} = 5144.75 \] \[ S_{jp} = 39\,950 - \frac{979\cdot 735}{20} = 3971.75 \]

b) \(r = \dfrac{3971.75}{\sqrt{4412.95\cdot 5144.75}} = \dfrac{3971.75}{4764.66} \approx 0.834\).

c) Сильная положительная корреляция → Нимер прав: сочные апельсины действительно стоят дороже.

Ответ

\[ \boxed{r \approx 0.834,\ \text{сильная положительная — Нимер прав}} \]

Two variables \(v, m\), coded by \(x = v - 30\), \(y = m/20\). Given \(\sum x = 287\), \(\sum x^{2} = 13\,879\), \(\sum y = 75.5\), \(\sum y^{2} = 627.25\), \(\sum xy = 2661\).

a Complete the coded table. b Calculate \(S_{xx},\ S_{yy},\ S_{xy}\). c PMCC for \(x, y\). d PMCC for \(v, m\). e Describe correlation.

Кодировка, потом \(S\) и \(r\). \(r\) для исходных данных.

a) Кодировка \(x = v-30\), \(y = m/20\) для всех 10 строк:

\(v\)507060824535110703530
\(x\)20403052155804050
\(m\)14020018021012010020018012060
\(y\)710910.56510963

b) \(n = 10\):

\[ S_{xx} = 13\,879 - \frac{287^{2}}{10} = 5642.1 \] \[ S_{yy} = 627.25 - \frac{75.5^{2}}{10} = 57.225 \] \[ S_{xy} = 2661 - \frac{287\cdot 75.5}{10} = 494.15 \]

Corr — Problem 8

Условие (Problem statement)

10 cows, additive \(x\) (25 gm units) and yield \(y\) (gallons). \(x: 1..10\); \(y: 4.0, 4.2, 4.3, 4.5, 4.5, 4.7, 5.2, 5.2, 5.1, 5.1\). Given \(S_{xx} = 28,\ S_{yy} = 0.90857,\ S_{xy} = 4.8\) (для первых 7 коров).

a) scatter; b) выводы; c) оптимальное количество добавки; d) почему H, I, J исключены; e) PMCC для 7 коров; f) как изменится \(r\) для всех 10.

Добавка vs удой. r для первых 7. Прогноз и интерпретация.

Решение (Solution)

a–b) Точки растут с \(x\), затем выходят на плато при \(x \ge 7\). Линейная связь верна только для \(x = 1\ldots 7\).

c) Чтобы максимизировать удой при минимуме добавки — выбираем \(x = 7\) (там удой уже почти максимальный, дальше прирост незначителен).

d) Коровы H, I, J (\(x = 8, 9, 10\)) исключены, потому что удой выходит на плато — связь перестаёт быть линейной.

e) Для 7 коров: \(r = \dfrac{4.8}{\sqrt{28\cdot 0.90857}} = \dfrac{4.8}{5.044} \approx 0.952\). Сильная положительная.

f) Для всех 10 коров \(r\) будет меньше, так как добавление трёх «нелинейных» точек ослабляет линейную связь.

Ответ

\[ \boxed{r_{7} \approx 0.952; \quad r_{10} < r_{7}; \quad \text{опт. } x \approx 7} \]

c) \(n = 10\):

\[ S_{cf} = 626\,100 - \frac{18\,200\cdot 365}{10} = 626\,100 - 664\,300 = -38\,200 \]

d) Удобная кодировка, упрощающая числа: \(x = c/100\), \(y = f\) (или \(y = f - 30\)). \(r\) не изменится.

Часть 2. Regression (Регрессия)

Reg — Problem 1

Условие (Problem statement)

Metal rod, \(l = 0.009t - 0.25\) (range \(30°\text{–}400°\text{C}\)).

a \(t = 300°\text{C}\). b \(t = 530°\text{C}\). c Reliability comments.

Регрессия удлинения от температуры. Прогноз и обсуждение.

Решение (Solution)

a) \(l = 0.009\cdot 300 - 0.25 = 2.7 - 0.25 = 2.45\) мм.

b) \(l = 0.009\cdot 530 - 0.25 = 4.77 - 0.25 = 4.52\) мм.

c) \(300°\text{C}\) находится внутри диапазона данных (\(30\text{–}400\)) — это интерполяция, надёжно. \(530°\text{C}\) — вне диапазона, это экстраполяция; модель могла перестать быть верной (например, металл повёл себя нелинейно), поэтому прогноз ненадёжен.

Ответ

\[ \boxed{\text{a) } 2.45 \text{ мм} \quad \text{b) } 4.52 \text{ мм} \quad \text{c) (a) надёжно, (b) ненадёжно}} \]

a) \(t = 1.96 + 0.950\,s\).

b) \(s = 50: \ t = 1.96 + 0.950\cdot 50 = 1.96 + 47.52 \approx 49.48\).

Ответ

\[ \boxed{\text{a) } t = 1.96 + 0.95s \quad \text{b) } t \approx 49.5} \]

Energy consumption \(x\) and GNP \(y\) for 8 countries. Compute regression \(y\) on \(x\) and predict.

\[ \boxed{y = 224 + 12.9x; \ y(100) \approx 1511; \ x(3500) \approx 255 \text{ (ненадёжно)}} \]

Reg — Problem 9

Условие (Problem statement)

7 days: items \(n\), weight \(w\) (kg). Coding \(x = n - 10,\ y = w/400\).

a) \(S_{xy},\ S_{xx}\); b) regression \(y\) on \(x\); c) regression \(w\) on \(n\); d) \(w\) for 20 items; e) почему 100 ненадёжно.

Кодировка → регрессия → раскодирование → прогнозы.

Решение (Solution)

Кодированные значения: \(x = n - 10\): 0, 3, 12, 5, 14, 6, 9. \(y = w/400\): 7, 9, 15, 9, 13, 11, 13.

\(\sum x = 49,\ \sum y = 77,\ \sum x^{2} = 491,\ \sum xy = 617\), \(n = 7\):

\[ S_{xx} = 491 - \frac{49^{2}}{7} = 148,\quad S_{xy} = 617 - \frac{49\cdot 77}{7} = 78 \]

a) \(S_{xx} = 148,\ S_{xy} = 78\).

b) \(b = 78/148 \approx 0.527,\ \bar{x} = 7,\ \bar{y} = 11\). \(a = 11 - 0.527\cdot 7 \approx 7.31\). \(y = 7.31 + 0.527x\).

c) Подставляем \(y = w/400,\ x = n - 10\):

\[ \frac{w}{400} = 7.31 + 0.527(n - 10) \Rightarrow w = 2924 + 210.8(n - 10) = 211n + 816 \]

d) \(n = 20: \ w \approx 211\cdot 20 + 816 = 4220 + 816 = 5036\) кг (точнее ≈ 5032).
