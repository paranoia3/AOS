Chi-squared Tests (Критерий хи-квадрат)

When to use it

Критерий согласия (chi-squared goodness-of-fit test) проверяет, насколько наблюдаемые частоты (observed frequencies, O) соответствуют ожидаемым частотам (expected frequencies, E), которые предсказывает некоторая модель (биномиальное, Пуассоновское, равномерное, нормальное распределение). Тот же критерий применяется к таблицам сопряжённости (contingency tables) для проверки независимости двух признаков.

Ключевые слова в условии:

  • «test whether the data can be modelled by a binomial / Poisson / uniform distribution» → goodness-of-fit;
  • «test whether there is association between» / «test for independence» → contingency table;
  • «at the 5% / 10% / 1% significance level» → уровень значимости \( \alpha \).

Definitions

Распределение хи-квадрат (chi-squared distribution) \( \chi^{2}_{\nu} \) — распределение суммы квадратов \( \nu \) независимых стандартных нормальных величин. Зависит от одного параметра — числа степеней свободы (degrees of freedom) \( \nu \).

Ожидаемая частота (expected frequency) \(E\) для категории — это сколько наблюдений мы бы получили в этой категории, если бы модель из \(H_{0}\) была верна: \( E_{i} = n \cdot p_{i} \) (для goodness-of-fit) или \( E_{ij} = \dfrac{\text{итог по строке} \times \text{итог по столбцу}}{\text{общий итог}} \) (для таблицы сопряжённости).

Key formulas

Формула 1 — Статистика \[ X^{2} = \sum_{i} \frac{(O_{i} - E_{i})^{2}}{E_{i}} \qquad \text{или эквивалентно} \qquad X^{2} = \sum_{i}\frac{O_{i}^{2}}{E_{i}} - N \]
Чем сильнее \(O\) отклоняется от \(E\), тем больше \(X^{2}\)
Формула 2 — Ожидаемая частота для контингентной таблицы \[ E_{ij} = \frac{(\text{строка } i)\cdot(\text{столбец } j)}{\text{общий итог}} \]
Формула 3 — Степени свободы \[ \nu = (\text{число клеток после объединения}) - (\text{число ограничений}) \]
Для таблицы \(h \times k\): \( \nu = (h-1)(k-1) \)

Degrees of freedom по типу модели

РаспределениеПараметры известныПараметры оцениваются
Discrete uniform\(n - 1\)—
Binomial\(n - 1\)\(n - 2\) (оценили \(p\))
Poisson\(n - 1\)\(n - 2\) (оценили \( \lambda \))
Normal\(n - 1\)\(n - 2\) или \(n - 3\) (1 или 2 параметра)
Continuous uniform\(n - 1\)—

Здесь \(n\) — число клеток после объединения. Каждое ограничение (известный итог и каждый оценённый параметр) уменьшает \( \nu \) на 1.

How to use it (step by step)

  1. Гипотезы. \( H_{0} \): данные согласуются с моделью / признаки независимы. \( H_{1} \): данные не согласуются / признаки связаны.
  2. Ожидаемые частоты. Вычисли \(E_{i}\) для каждой клетки.
  3. Объедини клетки с \(E_{i} < 5\) с соседними, пока все \(E_{i} \ge 5\).
  4. Считай статистику \( X^{2} = \sum (O - E)^{2}/E \).
  5. Степени свободы: \( \nu = (\text{клетки}) - 1 - (\text{число оценённых параметров}) \) или \( (h-1)(k-1) \) для контингентной таблицы.
  6. Критическое значение \( \chi^{2}_{\nu,\,\alpha} \) — из таблицы критических значений по \( \nu \) и уровню \( \alpha \).
  7. Вывод. Если \( X^{2} > \chi^{2}_{\nu,\,\alpha} \) — отклонить \(H_{0}\); иначе — принять.

Common mistakes

  • Не объединили клетки с \(E < 5\). Это даёт завышенную статистику и неверный df.
  • Забыли вычесть степень свободы за оценённый параметр. Если \(p\) или \(\lambda\) оценили из данных, df = (клетки) − 2, а не −1.
  • Перепутали наблюдаемые и ожидаемые. В числителе — \((O - E)^{2}\), в знаменателе — \(E\).
  • Критическая область — справа. Отклоняем \(H_{0}\), когда \(X^{2}\) большое.
  • Правильная формулировка гипотез. Для contingency — «независимы»/«связаны»; для согласия — «следует распределению X»/«не следует».

Mini example

Бросаем кубик 60 раз и получаем частоты: 8, 9, 12, 11, 10, 10 (грани 1–6). Проверим гипотезу о честности кубика на уровне 5%.

\(H_{0}\): кубик честный (равномерное распределение). \(E_{i} = 60/6 = 10\) для каждой грани.

\[ X^{2} = \frac{(8-10)^{2}}{10} + \frac{(9-10)^{2}}{10} + \frac{(12-10)^{2}}{10} + \frac{(11-10)^{2}}{10} + \frac{(10-10)^{2}}{10} + \frac{(10-10)^{2}}{10} \] \[ = 0.4 + 0.1 + 0.4 + 0.1 + 0 + 0 = 1.0 \]

\( \nu = 6 - 1 = 5 \). Критическое \( \chi^{2}_{5,\,0.05} = 11.07 \). Так как \(1.0 < 11.07\), принимаем \(H_{0}\): нет оснований сомневаться в честности кубика.
