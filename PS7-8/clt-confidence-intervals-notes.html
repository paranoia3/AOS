CLT & Confidence Intervals (ЦПТ и доверительные интервалы)

When to use it

Эта тема — про выборку (sample) и то, насколько надёжно по ней оценить среднее генеральной совокупности \( \mu \). Главных идей две:

  • Центральная предельная теорема (Central Limit Theorem, CLT) — если выборка большая, то среднее выборки распределено примерно нормально, даже если исходное распределение не нормальное;
  • Доверительный интервал (confidence interval, CI) — диапазон правдоподобных значений \( \mu \), построенный по выборке.

Ключевые слова в условии:

  • «a sample of size n is taken …, find the probability that the sample mean / total …» → ЦПТ;
  • «find a 95% / 90% / 99% confidence interval for the mean» → CI;
  • «find the sample size required so that the width is less than …» → подбор \(n\) из ширины CI.

Definitions

Несмещённая оценка (unbiased estimator) — оценка, чьё мат. ожидание равно оцениваемому параметру. \( \overline{X} = \dfrac{\sum X}{n} \) — несмещённая оценка \( \mu \). \( S^{2} = \dfrac{\sum X^{2} - n\overline{X}^{2}}{n-1} \) — несмещённая оценка \( \sigma^{2} \).

Стандартная ошибка среднего (standard error of the mean) — стандартное отклонение распределения выборочного среднего: \( \mathrm{SE}(\overline{X}) = \dfrac{\sigma}{\sqrt{n}} \).

Доверительный интервал (confidence interval) для \( \mu \) уровня \( 100(1 - \alpha)\% \) — интервал \( (\overline{x} - z\,\sigma/\sqrt{n},\ \overline{x} + z\,\sigma/\sqrt{n}) \), который в \(100(1-\alpha)\%\) случаев накроет истинное \( \mu \) при многократном повторении опыта.

Key formulas

Формула 1 — ЦПТ \[ \overline{X} \,\approx\, \mathrm{N}\!\left(\mu,\ \frac{\sigma^{2}}{n}\right) \quad \text{при больших } n \]
Среднее выборки распределено примерно нормально (с любым исходным распределением)
Формула 2 — Стандартизация \[ Z = \frac{\overline{X} - \mu}{\sigma / \sqrt{n}} \,\sim\, \mathrm{N}(0,1) \]
Для вероятностей задач на ЦПТ — стандартизуй среднее выборки
Формула 3 — Доверительный интервал для \( \mu \) \[ \overline{x} \pm z \cdot \frac{\sigma}{\sqrt{n}} \]
При известном \( \sigma \); \(z\) берётся из критических значений нормального распределения
Формула 4 — Сумма \[ T = \sum_{i=1}^{n} X_{i} \,\approx\, \mathrm{N}\!\left(n\mu,\ n\sigma^{2}\right) \]
Сумма \(n\) одинаковых независимых величин — нормальна при большом \(n\)

How to use it (step by step)

  1. Тип задачи: «вероятность среднего/суммы» → ЦПТ; «оцени \( \mu \) по выборке» → CI.
  2. ЦПТ. Запиши \( \overline{X} \sim \mathrm{N}(\mu, \sigma^{2}/n) \) (или \( T \sim \mathrm{N}(n\mu, n\sigma^{2}) \) для суммы), стандартизуй и используй таблицы \( \Phi \).
  3. CI. Найди \( \overline{x} \) (из суммы или из условия), бери \( z \) по уровню доверия, считай \( \overline{x} \pm z\sigma/\sqrt{n} \).
  4. Подбор \(n\) по ширине CI. Полная ширина \( w = 2z\sigma/\sqrt{n} \le w_{0} \Rightarrow n \ge \left(\dfrac{2z\sigma}{w_{0}}\right)^{2} \). Округлять \(n\) вверх.
  5. Дискретные распределения через ЦПТ. Для подсчётов (биномиальное, Пуассон) можно (часто без поправки на непрерывность на этом уровне) применять нормальное приближение к сумме.

Common mistakes

  • \( \sigma \) vs \( \sigma/\sqrt{n} \). В формуле ЦПТ используется именно стандартная ошибка \( \sigma/\sqrt{n} \), а не \( \sigma \).
  • Сумма vs среднее. Для суммы \(T\): дисперсия \(n\sigma^{2}\); для среднего \( \overline{X} \): дисперсия \( \sigma^{2}/n \). Перепутать — типичная ошибка.
  • Округление \(n\). Размер выборки всегда округляется вверх (если нужно «не менее»), потому что меньшее \(n\) не обеспечит условие.
  • Различие задач 90/95/99. Чем выше уровень доверия, тем больше \(z\) и шире CI.
  • «Объём выборки большой». Для применимости ЦПТ обычно достаточно \( n \ge 30 \) (если распределение не сильно скошенное).

Mini example

Из распределения с \( \mu = 50 \), \( \sigma = 10 \) взята выборка \( n = 100 \). Найдём \( P(\overline{X} > 52) \).

По ЦПТ \( \overline{X} \approx \mathrm{N}(50,\ 100/100) = \mathrm{N}(50,\ 1) \), \( \mathrm{SE} = 1 \).

\[ z = \frac{52 - 50}{1} = 2, \qquad P(\overline{X} > 52) \approx 1 - \Phi(2) = 1 - 0.9772 = 0.0228 \]
