Problem 1 (Discrete)

Условие (Problem statement)

The random variable \(X\) has probability function

\[ P(X = x) = \dfrac{x}{21}, \qquad x = 1, 2, 3, 4, 5, 6. \]

a Construct a table giving the probability distribution of \(X\).

Find b \(P(2 < X \leq 5)\),  c \(\mathrm{E}(X)\),  d \(\mathrm{Var}(X)\),  e \(\mathrm{Var}(3 - 2X)\).

Случайная величина \(X\) имеет функцию вероятности \( P(X=x)=x/21 \) при \( x=1,2,3,4,5,6 \). Построить таблицу распределения и найти указанные характеристики.

Дано (Given)

  • \( P(X = x) = \dfrac{x}{21} \), \( x \in \{1, 2, 3, 4, 5, 6\} \)

Найти (Find)

Таблицу распределения, \(P(2 < X \leq 5)\), \(\mathrm{E}(X)\), \(\mathrm{Var}(X)\), \(\mathrm{Var}(3-2X)\).

Решение (Solution)

(a) Таблица распределения вероятностей. Подставляем \( x = 1, 2, \ldots, 6 \) в формулу \( P(X=x)=x/21 \):

\(x\)123456
\(P(X=x)\)\(\tfrac{1}{21}\)\(\tfrac{2}{21}\)\(\tfrac{3}{21}\)\(\tfrac{4}{21}\)\(\tfrac{5}{21}\)\(\tfrac{6}{21}\)

Проверим нормировку: \( \sum P = \dfrac{1+2+3+4+5+6}{21} = \dfrac{21}{21} = 1 \) ✓

(b) \( P(2 < X \leq 5) \) включает значения \( X = 3, 4, 5 \) (не включает 2, но включает 5):

\[ P(2 < X \leq 5) = \dfrac{3}{21} + \dfrac{4}{21} + \dfrac{5}{21} = \dfrac{12}{21} = \dfrac{4}{7} \]

(c) Математическое ожидание по формуле \( \mathrm{E}(X) = \sum x \cdot P(X=x) \):

\[ \begin{aligned} \mathrm{E}(X) &= 1 \cdot \tfrac{1}{21} + 2 \cdot \tfrac{2}{21} + 3 \cdot \tfrac{3}{21} + 4 \cdot \tfrac{4}{21} + 5 \cdot \tfrac{5}{21} + 6 \cdot \tfrac{6}{21} \\ &= \tfrac{1 + 4 + 9 + 16 + 25 + 36}{21} \\ &= \tfrac{91}{21} = \tfrac{13}{3} \end{aligned} \]

(d) Дисперсия. Сначала \( \mathrm{E}(X^2) \):

\[ \begin{aligned} \mathrm{E}(X^2) &= 1 \cdot \tfrac{1}{21} + 4 \cdot \tfrac{2}{21} + 9 \cdot \tfrac{3}{21} + 16 \cdot \tfrac{4}{21} + 25 \cdot \tfrac{5}{21} + 36 \cdot \tfrac{6}{21} \\ &= \tfrac{1 + 8 + 27 + 64 + 125 + 216}{21} \\ &= \tfrac{441}{21} = 21 \end{aligned} \] \[ \mathrm{Var}(X) = \mathrm{E}(X^2) - [\mathrm{E}(X)]^2 = 21 - \left(\tfrac{13}{3}\right)^2 = 21 - \tfrac{169}{9} = \tfrac{189 - 169}{9} = \tfrac{20}{9} \]

(e) Используем свойство \( \mathrm{Var}(aX + b) = a^2 \mathrm{Var}(X) \) с \( a = -2 \):

\[ \mathrm{Var}(3 - 2X) = (-2)^2 \cdot \mathrm{Var}(X) = 4 \cdot \tfrac{20}{9} = \tfrac{80}{9} \approx 8.8889 \]

Ответ (Answer)

\[ \boxed{P(2 < X \leq 5) = \tfrac{4}{7}, \quad \mathrm{E}(X) = \tfrac{13}{3}, \quad \mathrm{Var}(X) = \tfrac{20}{9}, \quad \mathrm{Var}(3-2X) = \tfrac{80}{9}} \]

Problem 2 (Discrete)

Условие (Problem statement)

The discrete random variable \(X\) has the probability distribution shown.

\(x\)-2-10123
\(P(X=x)\)0.10.20.3\(r\)0.10.1

Find a \(r\),  b \(P(-1 \leq X < 2)\),  c \(F(0.6)\),  d \(\mathrm{E}(2X + 3)\),  e \(\mathrm{Var}(2X + 3)\).

Дискретная случайная величина \(X\) имеет распределение, заданное таблицей. Найти неизвестную вероятность \(r\) и указанные характеристики.

Дано (Given)

  • Таблица распределения с неизвестным \( r \).

Найти (Find)

\(r\), \(P(-1 \leq X < 2)\), \(F(0.6)\), \(\mathrm{E}(2X+3)\), \(\mathrm{Var}(2X+3)\).

Решение (Solution)

(a) Используем условие нормировки \( \sum P(X=x) = 1 \):

\[ 0.1 + 0.2 + 0.3 + r + 0.1 + 0.1 = 1 \] \[ 0.8 + r = 1 \implies r = 0.2 \]

(b) \( P(-1 \leq X < 2) \) включает \( X = -1, 0, 1 \) (включает \(-1\), не включает 2):

\[ P(-1 \leq X < 2) = 0.2 + 0.3 + 0.2 = 0.7 \]

(c) \( F(0.6) = P(X \leq 0.6) \). Так как \( X \) принимает целые значения, \( P(X \leq 0.6) = P(X \leq 0) \):

\[ F(0.6) = P(X = -2) + P(X = -1) + P(X = 0) = 0.1 + 0.2 + 0.3 = 0.6 \]

(d) Сначала найдём \( \mathrm{E}(X) \):

\[ \begin{aligned} \mathrm{E}(X) &= (-2)(0.1) + (-1)(0.2) + (0)(0.3) + (1)(0.2) + (2)(0.1) + (3)(0.1) \\ &= -0.2 - 0.2 + 0 + 0.2 + 0.2 + 0.3 = 0.3 \end{aligned} \]

По свойству линейности: \( \mathrm{E}(2X + 3) = 2 \mathrm{E}(X) + 3 = 2(0.3) + 3 = 3.6 \).

(e) Сначала найдём \( \mathrm{E}(X^2) \):

\[ \begin{aligned} \mathrm{E}(X^2) &= 4(0.1) + 1(0.2) + 0(0.3) + 1(0.2) + 4(0.1) + 9(0.1) \\ &= 0.4 + 0.2 + 0 + 0.2 + 0.4 + 0.9 = 2.1 \end{aligned} \] \[ \mathrm{Var}(X) = 2.1 - (0.3)^2 = 2.1 - 0.09 = 2.01 \] \[ \mathrm{Var}(2X + 3) = 2^2 \cdot \mathrm{Var}(X) = 4 \cdot 2.01 = 8.04 \]

Ответ (Answer)

\[ \boxed{r = 0.2,\; P(-1 \leq X < 2) = 0.7,\; F(0.6) = 0.6,\; \mathrm{E}(2X+3) = 3.6,\; \mathrm{Var}(2X+3) = 8.04} \]

Problem 3 (Discrete)

Условие (Problem statement)

A discrete random variable \(X\) has the probability distribution shown in the table below.

\(x\)012
\(P(X=x)\)\(\tfrac{1}{3}\)\(b\)\(\tfrac{1}{3} + b\)

a Find the value of \(b\).   b Show that \(\mathrm{E}(X) = 1.3\).   c Find the exact value of \(\mathrm{Var}(X)\).   d Find the exact value of \(P(X \leq 1.5)\).

Дискретная случайная величина \(X\) имеет распределение из таблицы. Найти \(b\), показать что \(\mathrm{E}(X)=1.3\), найти точную дисперсию и \(P(X \leq 1.5)\).

Дано (Given)

  • \( P(X=0) = \tfrac{1}{3} \), \( P(X=1) = b \), \( P(X=2) = \tfrac{1}{3} + b \)

Найти (Find)

\(b\), \(\mathrm{E}(X)\), \(\mathrm{Var}(X)\), \(P(X \leq 1.5)\).

Решение (Solution)

(a) Из условия нормировки:

\[ \tfrac{1}{3} + b + \tfrac{1}{3} + b = 1 \] \[ \tfrac{2}{3} + 2b = 1 \implies 2b = \tfrac{1}{3} \implies b = \tfrac{1}{6} \]

Тогда \( P(X=1) = \tfrac{1}{6} \), \( P(X=2) = \tfrac{1}{3} + \tfrac{1}{6} = \tfrac{2}{6} + \tfrac{1}{6} = \tfrac{3}{6} = \tfrac{1}{2} \).

(b) Считаем \( \mathrm{E}(X) \):

\[ \begin{aligned} \mathrm{E}(X) &= 0 \cdot \tfrac{1}{3} + 1 \cdot \tfrac{1}{6} + 2 \cdot \tfrac{1}{2} \\ &= 0 + \tfrac{1}{6} + 1 = \tfrac{1}{6} + \tfrac{6}{6} = \tfrac{7}{6} \\ &\approx 1.1\overline{6} \end{aligned} \]

Хм, получилось \( 7/6 \approx 1.167 \), а не 1.3.

В этом случае значение, объявленное в условии (\(\mathrm{E}(X) = 1.3\)), не сходится с прямым расчётом — наша таблица даёт \( \mathrm{E}(X) = \tfrac{7}{6} \). Видимо, в исходной задаче последний столбец таблицы был \( P(X=2) = \tfrac{1}{3} + 2b \) (а не \( \tfrac{1}{3} + b \)), что соответствует учебному ответу.

Проверим эту гипотезу: при \( P(X=2) = \tfrac{1}{3} + 2b \) нормировка даёт \( \tfrac{1}{3} + b + \tfrac{1}{3} + 2b = 1 \implies 3b = \tfrac{1}{3} \implies b = \tfrac{1}{9} \). Тогда \( P(X=2) = \tfrac{1}{3} + \tfrac{2}{9} = \tfrac{5}{9} \), и \( \mathrm{E}(X) = 0 + \tfrac{1}{9} + 2 \cdot \tfrac{5}{9} = \tfrac{11}{9} \approx 1.22 \). Тоже не 1.3.

Ниже даём решение строго по таблице из PDF.

(c) Точная дисперсия (на основе нашей таблицы):

\[ \mathrm{E}(X^2) = 0 \cdot \tfrac{1}{3} + 1 \cdot \tfrac{1}{6} + 4 \cdot \tfrac{1}{2} = \tfrac{1}{6} + 2 = \tfrac{13}{6} \] \[ \mathrm{Var}(X) = \tfrac{13}{6} - \left(\tfrac{7}{6}\right)^2 = \tfrac{13}{6} - \tfrac{49}{36} = \tfrac{78}{36} - \tfrac{49}{36} = \tfrac{29}{36} \]

(d) \( P(X \leq 1.5) = P(X = 0) + P(X = 1) = \tfrac{1}{3} + \tfrac{1}{6} = \tfrac{2}{6} + \tfrac{1}{6} = \tfrac{3}{6} = \tfrac{1}{2} \).

Ответ (Answer)

\[ \boxed{b = \tfrac{1}{6},\; \mathrm{E}(X) = \tfrac{7}{6},\; \mathrm{Var}(X) = \tfrac{29}{36},\; P(X \leq 1.5) = \tfrac{1}{2}} \]

\[ k + 0 + k + 2k = 4k = 1 \implies k = \tfrac{1}{4} \quad \checkmark \]

Итак, распределение:

\(x\)0123
\(P(X=x)\)\(\tfrac{1}{4}\)0\(\tfrac{1}{4}\)\(\tfrac{2}{4} = \tfrac{1}{2}\)

\[ \begin{aligned} \mathrm{E}(X) &= 0 \cdot \tfrac{1}{4} + 1 \cdot 0 + 2 \cdot \tfrac{1}{4} + 3 \cdot \tfrac{1}{2} \\ &= 0 + 0 + \tfrac{1}{2} + \tfrac{3}{2} = 2 \end{aligned} \]

Второй момент:

\[ \begin{aligned} \mathrm{E}(X^2) &= 0 \cdot \tfrac{1}{4} + 1 \cdot 0 + 4 \cdot \tfrac{1}{4} + 9 \cdot \tfrac{1}{2} \\ &= 0 + 0 + 1 + 4.5 = 5.5 \quad \checkmark \end{aligned} \]

\[ \mathrm{Var}(X) = \mathrm{E}(X^2) - [\mathrm{E}(X)]^2 = 5.5 - 4 = 1.5 \] \[ \mathrm{Var}(2X - 2) = 2^2 \cdot \mathrm{Var}(X) = 4 \cdot 1.5 = 6 \]

Ответ (Answer)

\[ \boxed{k = \tfrac{1}{4},\; \mathrm{E}(X) = 2,\; \mathrm{Var}(2X - 2) = 6} \]

Problem 6 (Discrete)

Условие (Problem statement)

A discrete random variable is such that each of its values is assumed to be equally likely.

a Write the name of the distribution.  b Give an example of such a distribution.

A discrete random variable \(X\) is defined above can take values \(0, 1, 2, 3\) and \(4\).

Find c \(\mathrm{E}(X)\),  d \(\mathrm{Var}(X)\).

Дискретная случайная величина такова, что все её значения равновероятны. (a) Назвать распределение. (b) Привести пример. Затем \(X\) принимает значения \(0, 1, 2, 3, 4\). Найти \(\mathrm{E}(X)\) и \(\mathrm{Var}(X)\).

Дано (Given)

  • \(X\) принимает значения \(0, 1, 2, 3, 4\), все с равной вероятностью \( \tfrac{1}{5} \)

Найти (Find)

Название распределения, пример, \(\mathrm{E}(X)\), \(\mathrm{Var}(X)\).

Решение (Solution)

(a) Название. Распределение, в котором все значения равновероятны, называется дискретным равномерным распределением (discrete uniform distribution).

(b) Пример. Бросание правильной игральной кости: \( X \in \{1, 2, 3, 4, 5, 6\} \), каждая грань с вероятностью \( \tfrac{1}{6} \).

(c) Математическое ожидание. Для \(X\) на значениях \(0, 1, 2, 3, 4\) с равными вероятностями \( \tfrac{1}{5} \):

\[ \mathrm{E}(X) = \tfrac{1}{5}(0 + 1 + 2 + 3 + 4) = \tfrac{10}{5} = 2 \]

(d) Дисперсия. Сначала \( \mathrm{E}(X^2) \):

\[ \mathrm{E}(X^2) = \tfrac{1}{5}(0 + 1 + 4 + 9 + 16) = \tfrac{30}{5} = 6 \] \[ \mathrm{Var}(X) = 6 - 2^2 = 6 - 4 = 2 \]

(Альтернативно: для равномерного распределения на \( \{0,1,\ldots,n\} \) — это сдвиг распределения на \( \{1, \ldots, n+1\} \); прямой расчёт быстрее.)

Ответ (Answer)

\[ \boxed{\text{Discrete uniform distribution},\; \mathrm{E}(X) = 2,\; \mathrm{Var}(X) = 2} \]

A random variable \(X\) has probability density function \(f(x)\) given by

Эскиз PDF, моду, CDF, \(P(0.5 < X < 1.5)\).

Решение (Solution)

Проверка нормировки:

\[ \int_{-2}^2 \tfrac{3}{32}(4 - x^2)\,dx = \tfrac{3}{32}\left[4x - \tfrac{x^3}{3}\right]_{-2}^2 = \tfrac{3}{32}\left((8 - \tfrac{8}{3}) - (-8 + \tfrac{8}{3})\right) = \tfrac{3}{32} \cdot \tfrac{32}{3} = 1 \quad \checkmark \]

(b) Мода. Максимум \(f\) — при \(x = 0\), значит:

\[ \text{Mode} = 0 \]

Полная запись:

\[ F(x) = \begin{cases} 0, & x < -2 \\ \dfrac{16 + 12x - x^3}{32}, & -2 \leq x \leq 2 \\ 1, & x > 2 \end{cases} \]

(d)

Problem 8 (Continuous)

Условие (Problem statement)

A random variable \(X\) has probability density function \(f(x)\) given by

\[ f(x) = \begin{cases} \tfrac{1}{3}, & 0 \leq x < 1 \\ \tfrac{2}{7}x^2, & 1 \leq x \leq 2 \\ 0, & \text{otherwise.} \end{cases} \]

a Find \(\mathrm{E}(X)\).  b Specify fully the cumulative distribution function of \(X\).  c Find the median of \(X\).

\(X\) имеет кусочную PDF (1/3 на \([0,1)\) и \(2x^2/7\) на \([1,2]\)). Найти \(\mathrm{E}(X)\), CDF и медиану.

Дано (Given)

  • \(f(x) = 1/3\) на \([0, 1)\)
  • \(f(x) = \tfrac{2}{7}x^2\) на \([1, 2]\)

Найти (Find)

\(\mathrm{E}(X)\), \(F(x)\), медиану.

Решение (Solution)

Проверка нормировки:

\[ \int_0^1 \tfrac{1}{3}\,dx + \int_1^2 \tfrac{2}{7}x^2\,dx = \tfrac{1}{3} + \tfrac{2}{7}\cdot\tfrac{x^3}{3}\Big|_1^2 = \tfrac{1}{3} + \tfrac{2}{21}(8-1) = \tfrac{1}{3} + \tfrac{14}{21} = \tfrac{1}{3} + \tfrac{2}{3} = 1 \quad \checkmark \]

(a) Математическое ожидание:

\[ \begin{aligned} \mathrm{E}(X) &= \int_0^1 x \cdot \tfrac{1}{3}\,dx + \int_1^2 x \cdot \tfrac{2}{7}x^2\,dx \\ &= \tfrac{1}{3}\cdot\tfrac{x^2}{2}\Big|_0^1 + \tfrac{2}{7}\cdot\tfrac{x^4}{4}\Big|_1^2 \\ &= \tfrac{1}{6} + \tfrac{2}{7}\cdot\tfrac{16 - 1}{4} = \tfrac{1}{6} + \tfrac{2}{7}\cdot\tfrac{15}{4} \\ &= \tfrac{1}{6} + \tfrac{30}{28} = \tfrac{1}{6} + \tfrac{15}{14} \\ &= \tfrac{7}{42} + \tfrac{45}{42} = \tfrac{52}{42} = \tfrac{26}{21} \approx 1.2381 \end{aligned} \]

(b) CDF. Для \(0 \leq x < 1\):

\[ F(x) = \int_0^x \tfrac{1}{3}\,dt = \tfrac{x}{3} \]

Для \(1 \leq x \leq 2\):

\[ F(x) = \tfrac{1}{3} + \int_1^x \tfrac{2}{7}t^2\,dt = \tfrac{1}{3} + \tfrac{2}{21}(x^3 - 1) = \tfrac{7}{21} + \tfrac{2x^3 - 2}{21} = \tfrac{2x^3 + 5}{21} \]

Полностью:

\[ F(x) = \begin{cases} 0, & x < 0 \\ \tfrac{x}{3}, & 0 \leq x < 1 \\ \tfrac{2x^3 + 5}{21}, & 1 \leq x \leq 2 \\ 1, & x > 2 \end{cases} \]

Проверка: \(F(1) = \tfrac{2+5}{21} = \tfrac{7}{21} = \tfrac{1}{3}\) ✓; \(F(2) = \tfrac{16+5}{21} = 1\) ✓.

(c) Медиана. Так как \(F(1) = \tfrac{1}{3} < 0.5\), медиана лежит в \([1, 2]\). Решаем:

\[ \tfrac{2m^3 + 5}{21} = 0.5 \implies 2m^3 + 5 = 10.5 \implies 2m^3 = 5.5 \implies m^3 = 2.75 \] \[ m = \sqrt[3]{2.75} \approx 1.4015 \]

Ответ (Answer)

\[ \boxed{\mathrm{E}(X) = \tfrac{26}{21} \approx 1.2381,\; F(x) = \begin{cases} x/3, & 0 \leq x < 1 \\ (2x^3+5)/21, & 1 \leq x \leq 2 \end{cases},\; m = \sqrt[3]{2.75} \approx 1.4015} \]

Problem 9 (Continuous)

Условие (Problem statement)

A continuous random variable \(X\) has probability density function \(f(x)\) given by

\[ f(x) = \begin{cases} kx - k, & 1 \leq x \leq 3 \\ 0, & \text{otherwise.} \end{cases} \]

where \(k\) is a positive constant.

a Show that \(k = \tfrac{1}{2}\).  b Find \(\mathrm{E}(X)\).  c Work out the cumulative distribution function, \(F(x)\).  d Show that the median value lies between 2.4 and 2.5.

\(X\) имеет PDF \(f(x) = k(x-1)\) на \([1,3]\). Показать что \(k=1/2\), найти \(\mathrm{E}(X)\), CDF и показать, что медиана между 2.4 и 2.5.

Дано (Given)

  • \(f(x) = k(x - 1)\) на \([1, 3]\), \(k > 0\)

Найти (Find)

\(k\), \(\mathrm{E}(X)\), \(F(x)\), показать, что медиана \(\in (2.4, 2.5)\).

Решение (Solution)

(a) Из нормировки:

\[ \int_1^3 k(x - 1)\,dx = k\left[\tfrac{(x-1)^2}{2}\right]_1^3 = k \cdot \tfrac{4}{2} = 2k = 1 \implies k = \tfrac{1}{2} \quad \checkmark \]

(b) Мат. ожидание:

\[ \begin{aligned} \mathrm{E}(X) &= \int_1^3 x \cdot \tfrac{1}{2}(x - 1)\,dx = \tfrac{1}{2}\int_1^3 (x^2 - x)\,dx \\ &= \tfrac{1}{2}\left[\tfrac{x^3}{3} - \tfrac{x^2}{2}\right]_1^3 \\ &= \tfrac{1}{2}\left(\left(9 - \tfrac{9}{2}\right) - \left(\tfrac{1}{3} - \tfrac{1}{2}\right)\right) \\ &= \tfrac{1}{2}\left(\tfrac{9}{2} - \left(-\tfrac{1}{6}\right)\right) = \tfrac{1}{2}\left(\tfrac{27}{6} + \tfrac{1}{6}\right) \\ &= \tfrac{1}{2} \cdot \tfrac{28}{6} = \tfrac{14}{6} = \tfrac{7}{3} \end{aligned} \]

(c) CDF для \(1 \leq x \leq 3\):

\[ F(x) = \int_1^x \tfrac{1}{2}(t - 1)\,dt = \tfrac{1}{2}\cdot\tfrac{(t-1)^2}{2}\Big|_1^x = \tfrac{(x - 1)^2}{4} \]

Полностью:

\[ F(x) = \begin{cases} 0, & x < 1 \\ \tfrac{(x-1)^2}{4}, & 1 \leq x \leq 3 \\ 1, & x > 3 \end{cases} \]

(d) Проверим значения функции в точках 2.4 и 2.5:

\[ F(2.4) = \tfrac{(1.4)^2}{4} = \tfrac{1.96}{4} = 0.49 < 0.5 \] \[ F(2.5) = \tfrac{(1.5)^2}{4} = \tfrac{2.25}{4} = 0.5625 > 0.5 \]

Поскольку \(F\) непрерывна и возрастает, существует единственное \(m \in (2.4, 2.5)\), такое что \(F(m) = 0.5\). ✓

Точное значение: \(\tfrac{(m-1)^2}{4} = 0.5 \implies (m-1)^2 = 2 \implies m = 1 + \sqrt{2} \approx 2.4142\).

Ответ (Answer)

\[ \boxed{k = \tfrac{1}{2},\; \mathrm{E}(X) = \tfrac{7}{3},\; F(x) = \tfrac{(x-1)^2}{4} \text{ на } [1, 3]} \] \[ \boxed{m = 1 + \sqrt{2} \approx 2.4142 \in (2.4,\, 2.5) \quad \checkmark} \]

Problem 10 (Continuous)

Условие (Problem statement)

The continuous random variable \(X\) has probability density function given by

\[ f(x) = \begin{cases} x, & 0 \leq x < 1 \\ \tfrac{3x^2}{14}, & 1 \leq x \leq 2 \\ 0, & \text{otherwise.} \end{cases} \]

a Sketch the probability density function of \(X\).  b Find the mode of \(X\).  c Find \(\mathrm{E}(2X)\).  d Find \(\mathrm{Var}(2X + 1)\).  e Specify fully the cumulative distribution function of \(X\).  f Using your answer to part e find the median of \(X\).

\(X\) имеет кусочную PDF: \(x\) на \([0,1)\) и \(3x^2/14\) на \([1,2]\). Нарисовать график, найти моду, \(\mathrm{E}(2X)\), \(\mathrm{Var}(2X+1)\), CDF и медиану.

Дано (Given)

  • \(f(x) = x\) на \([0, 1)\)
  • \(f(x) = \tfrac{3x^2}{14}\) на \([1, 2]\)

Найти (Find)

Эскиз PDF, моду, \(\mathrm{E}(2X)\), \(\mathrm{Var}(2X+1)\), \(F(x)\), медиану.

Решение (Solution)

Проверка нормировки:

\[ \int_0^1 x\,dx + \int_1^2 \tfrac{3x^2}{14}\,dx = \tfrac{1}{2} + \tfrac{3}{14}\cdot\tfrac{x^3}{3}\Big|_1^2 = \tfrac{1}{2} + \tfrac{1}{14}(8-1) = \tfrac{1}{2} + \tfrac{1}{2} = 1 \quad \checkmark \]

(a) Эскиз. На \([0, 1)\) график — прямая \(f(x) = x\) от \((0, 0)\) до \((1, 1)\). При \(x = 1\) происходит скачок вниз до \(f(1) = \tfrac{3}{14} \approx 0.214\), затем плавный рост по параболе \(\tfrac{3x^2}{14}\) до \(f(2) = \tfrac{12}{14} = \tfrac{6}{7} \approx 0.857\):

(b) Мода. На \([0, 1)\) \(f\) растёт до значения 1 (при \(x \to 1^-\)). На \([1, 2]\) максимум \(f(2) = \tfrac{6}{7} < 1\). Значит максимум плотности достигается при \(x = 1\) (точнее, в пределе слева):

\[ \text{Mode} = 1 \]

(c) \(\mathrm{E}(2X)\). Сначала \(\mathrm{E}(X)\):

\[ \begin{aligned} \mathrm{E}(X) &= \int_0^1 x \cdot x\,dx + \int_1^2 x \cdot \tfrac{3x^2}{14}\,dx \\ &= \int_0^1 x^2\,dx + \tfrac{3}{14}\int_1^2 x^3\,dx \\ &= \tfrac{1}{3} + \tfrac{3}{14}\cdot\tfrac{x^4}{4}\Big|_1^2 \\ &= \tfrac{1}{3} + \tfrac{3}{14}\cdot\tfrac{15}{4} = \tfrac{1}{3} + \tfrac{45}{56} \\ &= \tfrac{56}{168} + \tfrac{135}{168} = \tfrac{191}{168} \approx 1.1369 \end{aligned} \] \[ \mathrm{E}(2X) = 2 \mathrm{E}(X) = \tfrac{382}{168} = \tfrac{191}{84} \approx 2.2738 \]

(d) \(\mathrm{Var}(2X + 1)\). Сначала \(\mathrm{E}(X^2)\):

\[ \begin{aligned} \mathrm{E}(X^2) &= \int_0^1 x^3\,dx + \int_1^2 x^2 \cdot \tfrac{3x^2}{14}\,dx \\ &= \tfrac{1}{4} + \tfrac{3}{14}\int_1^2 x^4\,dx \\ &= \tfrac{1}{4} + \tfrac{3}{14}\cdot\tfrac{x^5}{5}\Big|_1^2 \\ &= \tfrac{1}{4} + \tfrac{3}{14}\cdot\tfrac{31}{5} = \tfrac{1}{4} + \tfrac{93}{70} \\ &= \tfrac{35}{140} + \tfrac{186}{140} = \tfrac{221}{140} \approx 1.5786 \end{aligned} \] \[ \mathrm{Var}(X) = \tfrac{221}{140} - \left(\tfrac{191}{168}\right)^2 \approx 1.5786 - 1.2926 \approx 0.2860 \]

Точное значение:

\[ \mathrm{Var}(X) = \tfrac{221}{140} - \tfrac{36481}{28224} = \tfrac{222768 - 182405}{141120} = \tfrac{40363}{141120} \approx 0.2860 \] \[ \mathrm{Var}(2X + 1) = 4 \mathrm{Var}(X) = \tfrac{161452}{141120} = \tfrac{40363}{35280} \approx 1.1440 \]

(e) CDF. Для \(0 \leq x < 1\):

\[ F(x) = \int_0^x t\,dt = \tfrac{x^2}{2} \]

Для \(1 \leq x \leq 2\):

\[ F(x) = \tfrac{1}{2} + \int_1^x \tfrac{3t^2}{14}\,dt = \tfrac{1}{2} + \tfrac{1}{14}(x^3 - 1) = \tfrac{7 + x^3 - 1}{14} = \tfrac{x^3 + 6}{14} \]

Полностью:

\[ F(x) = \begin{cases} 0, & x < 0 \\ \tfrac{x^2}{2}, & 0 \leq x < 1 \\ \tfrac{x^3 + 6}{14}, & 1 \leq x \leq 2 \\ 1, & x > 2 \end{cases} \]

Проверка: \(F(1) = \tfrac{1+6}{14} = \tfrac{1}{2}\) ✓; \(F(2) = \tfrac{14}{14} = 1\) ✓.

(f) Медиана. Заметим: \(F(1) = \tfrac{1}{2} = 0.5\) точно. Значит:

\[ m = 1 \]

Ответ (Answer)

\[ \boxed{\text{Mode} = 1,\; \mathrm{E}(2X) = \tfrac{191}{84} \approx 2.2738,\; \mathrm{Var}(2X+1) \approx 1.1440} \] \[ \boxed{F(x) = \begin{cases} x^2/2, & 0 \leq x < 1 \\ (x^3+6)/14, & 1 \leq x \leq 2 \end{cases},\quad m = 1} \]
